# COAI Website UI Kit

Interactive prototype of the Chaotically Organized AI marketing website.

## Pages / Screens
1. **Home (Landing Page)** — Hero, stat banner, packages, testimonials, final CTA
2. **Pricing** — Package ladder with feature lists
3. **Intake / Diagnostic** — Multi-step intake wizard
4. **JAX Sentinel** — Sub-product landing page

## Components
- `Nav.jsx` — Ticker + sticky nav bar with mobile drawer
- `Hero.jsx` — Hero section with mesh background, stats, CTA group
- `Packages.jsx` — Service card grid (3-up), package cards
- `Testimonials.jsx` — Testimonial card grid with result badges
- `Footer.jsx` — Multi-column footer with brand info

## Design Width
1280px desktop · 375px mobile (responsive)
